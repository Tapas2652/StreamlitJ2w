## 🚀 How to Use

### Daily Usage (Template)
Each morning, open the HTML file in a text editor and update:

```
1. Date          → Change "Monday, 02 March 2026" to today's date
2. Joiners Table → Update candidate names, client, POD, role, DOJ
3. POD Headcount → Update the number for PODs that received new joiners
4. Add ▲ badge  → Add the "+1 Today" badge + bounce arrow to relevant PODs
5. KPI Cards     → Update "Today's Joiners" count and "MTD Onboardings"
6. Total Bar     → Update the total headcount (3,168 → new total)
```

### Open in Browser
```bash
# Simply double-click the file, or:
open JTW_Daily_Onboarding_Dashboard_Responsive_02Mar2026.html        # macOS
start JTW_Daily_Onboarding_Dashboard_Responsive_02Mar2026.html       # Windows
xdg-open JTW_Daily_Onboarding_Dashboard_Responsive_02Mar2026.html   # Linux
```

### Send as Email
1. Open the `.html` file in a browser
2. Select All (`Ctrl+A`) → Copy (`Ctrl+C`)
3. Paste directly into Outlook / Gmail compose window
4. Send — all formatting and animations will render in modern email clients
